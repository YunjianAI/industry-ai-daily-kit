# Codex 启动 Prompt

把下面这段复制到你的 Codex 里：

```md
我想搭建一套“行业 AI 情报日报系统”。

请你先不要直接生成日报。先采访我，问清楚我的行业、关注方向、信息源和输出要求，然后再帮我生成第一版配置。

这套日报系统的基础来源包括：

1. AIHOT baseline scan
2. X / Twitter 搜索
3. Reddit 搜索
4. 知乎搜索
5. 官方政策和公告更新
6. 通用 WebSearch 和其他来源
7. 跨平台观察
8. 昨日漏项回扫

重要规则：

1. AIHOT 只作为 baseline scan，不作为最终证据源。
2. 大 AI 新闻需要保留，比如 OpenAI、Anthropic、Google、Meta、国产大模型等。
3. 日报重点要偏向我的行业方向，而不是泛泛 AI 新闻。
4. 每条重要信息都要尽量追溯到原始来源。
5. 不要把营销稿、重复新闻和低质量自媒体当成核心信息。
6. 如果平台抓不到有效内容，保留栏目并说明当天没有捕捉到有效内容。

请按顺序问我这些问题：

1. 我的行业是什么？
2. 我最关心 AI 在这个行业里的哪些方向？
3. 哪些大 AI 新闻我必须每天看到？
4. 哪些行业公司、竞品、产品、机构、专家需要重点盯？
5. 哪些平台必须纳入？
6. 日报是给谁看的？
7. 我希望日报多长？
8. 我希望每天固定有哪些栏目？
9. 哪些信息不要收？
10. 输出到哪里？

问完以后，请生成一套“我的行业 AI 情报日报配置”，包括：

1. 信息源清单
2. 关键词词库
3. 排除词清单
4. 日报栏目模板
5. 每日运行流程
6. 证据优先级规则
7. 第一版日报样例

先一问一答，不要一次问太多问题。
```
