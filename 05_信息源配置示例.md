# 信息源配置示例

下面是示例配置，不代表最终配置。使用者需要根据自己的行业重新填写。

```yaml
profile:
  industry: "待填写"
  audience: "自己 / 团队 / 老板 / 客户 / 内容选题"
  language: "zh-CN"
  timezone: "Asia/Shanghai"

ai_baseline:
  keep_big_ai_news: true
  providers:
    - OpenAI
    - Anthropic
    - Google DeepMind
    - Meta AI
    - Microsoft AI
    - NVIDIA
    - Alibaba Qwen
    - DeepSeek
    - Moonshot
    - Zhipu AI
  aihot_role: "baseline_only"

platforms:
  x_twitter:
    enabled: true
    purpose: "海外一手事件和从业者即时反应"
    fallback: "WebSearch site:x.com"
  reddit:
    enabled: true
    purpose: "开发者和用户真实反馈"
    suggested_subreddits:
      - LocalLLaMA
      - MachineLearning
      - artificial
      - singularity
    fallback: "WebSearch site:reddit.com"
  zhihu:
    enabled: true
    purpose: "中文长文和行业内观点"
    fallback: "WebSearch site:zhihu.com"
  official_sources:
    enabled: true
    purpose: "政策、公告、公司官方发布"
  general_web:
    enabled: true
    purpose: "行业媒体、公司博客、研究报告、产品发布"

keywords:
  must_track:
    - "待填写行业词"
    - "待填写产品词"
    - "待填写竞品词"
  ai_application_terms:
    - "AI agent"
    - "workflow automation"
    - "RAG"
    - "multimodal"
    - "voice agent"
    - "AI search"
  policy_terms:
    - "监管"
    - "合规"
    - "数据安全"
    - "算法备案"

exclude_terms:
  - "软文"
  - "招商加盟"
  - "标题党"
  - "无来源转载"

evidence_priority:
  high:
    - "公司官网"
    - "官方博客"
    - "政策原文"
    - "论文原文"
    - "产品文档"
  medium:
    - "X/Twitter 一手账号"
    - "Reddit 高互动帖子"
    - "知乎高质量长答"
    - "主流媒体"
  low:
    - "二手转述"
    - "营销号"
    - "无链接摘要"
```
